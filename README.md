# 🌸 Lavender Scentsy – E-commerce Perfume Store

Lavender Scentsy is a full-stack e-commerce web application built for a perfume brand. It provides a smooth shopping experience for customers while giving administrators full control over products, orders, and users.

The project is structured into three main parts:
- Client app – for customers
- Admin app – for store management
- Server – handles API, database, and business logic

---

## 🧠 What This Project Is About

Lavender Scentsy is designed to simulate a real-world online perfume store where users can browse fragrances, place orders, and manage their accounts.

On the other side, administrators can manage inventory, track orders, and control the overall store operations from a dedicated dashboard.

---

## ❗ Problem It Solves

Many small perfume businesses struggle with:
- Not having an online presence
- Difficulty managing orders manually
- Poor customer experience when buying products
- Lack of centralized control for products and users

This project addresses those issues by creating a digital platform that:
- Makes products accessible online
- Automates order management
- Improves user experience
- Gives admins full control in one place

---

## 💡 How It Solves the Problem

Lavender Scentsy solves these problems through a well-structured system:

### 🛍️ Customer (Client App)
- Browse available perfumes
- View product details
- Add items to cart
- Place orders
- View order history

### 🛠️ Admin Dashboard (Admin App)
- Add, edit, and delete products
- View all orders
- Check order details
- Manage inventory efficiently

### ⚙️ Backend (Server)
- Handles API requests
- Stores and retrieves data from the database
- Manages authentication and business logic
- Connects both client and admin apps seamlessly
